Ceci est une application desktop utilisée dans le cadre des épreuves du BTS SIO. Elle répond à des besoins énoncés dans un scénario fictif. Il ne s'agit pas d'une application répondant à des besoins réels ou 
utilisable dans un cadre professionnel.

Cahier des charges de l’application : 
Application desktop permettant de gérer le stock de chaussettes de l’entreprise Flocks. Cette application est en lien avec une API.
On y retrouve les fonctionnalités suivantes : 
FrontOffice :
 - F1 : Authentification avec token
 - F2 : Tableau de bord affichant les statistiques du mois en cours (pas réalisée)
 - F3 : Tableau de bord répertoriant les demandes de commandes + possibilité d’accepter / refuser
 - F4 : Formulaire permettant de sélectionner une paire de chaussette et de choisir les réparations à faire (pas réalisée)
 - F5 : Formulaire permettant de choisir une commande et l’agence qui va réceptionner la commande afin de la renvoyer vers le client
BackOffice : 
 - F1 : Tableau de bord permettant de gérer les utilisateurs de l’application (pas réalisée)
